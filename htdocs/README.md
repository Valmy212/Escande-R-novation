# Z Rénovation - Site vitrine

Ce projet est une structure de site vitrine inspirée de zrenovation.ch, développée en HTML, CSS et JavaScript.

## Structure

- `index.html` : page d'accueil
- `pages/services.html` : nos services
- `pages/realisations.html` : nos réalisations
- `pages/contact.html` : formulaire de contact
- `css/style.css` : styles globaux
- `js/main.js` : scripts JS
- `images/` : images du site

## Utilisation

Ouvre `index.html` dans ton navigateur pour voir le site.
